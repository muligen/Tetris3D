## 技术亮点

### 1. 高效的 FPS 计算
使用 `useFrame` 获取精确的时钟时间，而不是依赖 `performance.now()`，确保与 R3F 渲染周期同步。

### 2. 节流更新
每 250ms 更新一次显示，避免每帧都更新 DOM，大大降低开销。

### 3. Ref 优化
使用 `useRef` 存储帧历史和计算状态，避免触发组件重新渲染。

### 4. 动态颜色
根据 FPS 值动态选择颜色（绿色 >= 50, 黄色 >= 30, 红色 < 30），提供直观的性能反馈。

### 5. 非侵入式设计
- HTML overlay 而非 3D 元素
- pointerEvents: 'none' 不阻挡交互
- 默认隐藏，用户可选
