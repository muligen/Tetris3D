# 文章配图方案生成器

生成文章完整配图方案，并通过Unsplash搜索配图。

## 使用场景

当用户需要为一篇文章生成配图方案并获取图片时使用。

## 工作流程

### 步骤1：通读文章

读取用户指定的文章文件，理解文章主题、逻辑结构和阅读节奏。

### 步骤2：生成配图方案

严格按照以下要求生成配图方案：

1. **标记配图位置**：按文章逻辑和阅读节奏，标记出最佳配图位置，标注对应原文段落
2. **生成关键词**：每个配图位置生成1组精准关键词，用于Unsplash图库搜图
3. **明确规格**：每张图明确标准尺寸、核心画面描述、统一风格，确保和对应段落内容100%匹配
4. **控制数量**：
   - 1000字配2-3张
   - 2000字配3-5张
   - 3000字以上配5-8张
   - 配图间隔不低于300字，不打断阅读节奏

### 步骤3：输出配图方案

按以下格式输出：

```
## 一、文章概览与配图规划
- 文章字数
- 建议配图数量
- 整体风格定位

## 二、配图点位详细方案

### 【配图N】位置描述
- **对应原文段落**：引用原文
- **核心画面**：描述
- **标准尺寸**：宽×高
- **Unsplash搜图关键词**：英文关键词

## 三、配图位置一览表
| 序号 | 位置 | 对应段落关键词 | 与上张图间隔 |

## 四、风格统一规范
- 主色调、辅助色、图标风格、构图规范
```

### 步骤4：搜索Unsplash图片

对每个配图点位，使用WebFetch访问Unsplash搜索对应关键词，获取高质量图片链接：

```
https://unsplash.com/s/photos/{关键词}
```

### 步骤5：输出最终配图结果

为每个配图位置输出：
- 图片标题
- Unsplash图片链接（选择最匹配的2-3张供用户选择）
- 作者信息（用于署名）

## 输出规范

- 只输出方案和图片链接，不额外解释、不废话
- 图片需可商用（Unsplash图片默认可商用）
- 关键词用英文，便于Unsplash搜索
- 风格统一，确保整套配图视觉一致
